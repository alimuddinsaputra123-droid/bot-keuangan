💾 Backup & Restore (JSON)

💾

Backup Data

Simpan seluruh database ke file JSON. Gunakan untuk backup berkala atau migrasi data.

📤

Restore Data

Muat data dari file JSON backup. Data yang ada akan ditambahkan (tidak ditimpa).

⚠️ Catatan: File JSON berisi data mentah. Jangan edit manual kecuali Anda tahu formatnya. Selalu backup sebelum import data baru.
